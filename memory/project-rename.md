The project was renamed from **pyMC** to **openHop** due to a trademark conflict.

**Why:** Trademark issue required the name change. The old repo was at `archive/pyMC-RepeaterUI`; the new repo is `openHop_RepeaterUI`.

**How to apply:** Always use "openHop" in code, documentation, and conversation. Never refer to "pyMC" as the current product name. The backend was previously called `pyMC_Repeater`; the new equivalent is the `openHop` Python backend. The `pymc_core` Python module has been renamed to `openhop_core`.
